# Agenda de Técnicos – Acessanet Telecom

Sistema para gerenciar a agenda dos técnicos da **Acessanet Telecom**, permitindo acompanhar horários, clientes e status das ordens de serviço.

## Funcionalidades

- Visualizar horários de atendimento dos técnicos
- Consultar informações dos clientes agendados
- Acompanhar o status das ordens de serviço (pendente, em execução, concluída)

## Tecnologias

- Frontend: Html, css e JavaScript
- Backend: Node.js + Express

## Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/matheusdevsilva/agenda-tecnicos-acessanet.git
```
2. Instale as dependências:
```bash
cd api && npm install
```

3. Execute o backend:
```bash
npm run dev
```
4. Acesse http://localhost:12000/agenda-tecnicos.html ou http://{IP}:12000/agenda-tecnicos.html
